## Problemas criticos

### 1. Stripe NO configurado — $0 revenue posible
- Endpoint `/api/checkout` existe pero falla sin API keys
- Webhook existe pero no funciona sin secret
- Pagina de success tiene confetti pero es inalcanzable
- **Fix:** 30 min — crear producto en Stripe, poner keys en Vercel

### 2. Usuarios in-memory — datos se pierden
- `Map()` en auth.ts — cada restart de Railway borra todos los usuarios
- **Fix:** 4-6h — migrar a SQLite o PostgreSQL

### 3. Passwords en texto plano
```typescript
password, // In production, hash this!
```
- bcryptjs ya esta en package.json pero no se usa
- **Fix:** 30 min

### 4. Sin dominio propio
- Nadie paga $9.99/mes a una URL de Vercel temporal
- **Fix:** 15 min — apuntar dominio a Vercel

### 5. Hardcoded secret de NextAuth
```typescript
secret: process.env.NEXTAUTH_SECRET || 'chilljobs-secret-key-change-in-production'
```
- Session hijacking posible
- **Fix:** 5 min — poner variable de entorno en Vercel

### 6. CORS abierto (`allow_origins=["*"]`)
- Cualquiera puede acceder a la API
- Aceptable por ahora (datos publicos), pero cerrar cuando haya auth real

### 7. No-phone detection es keyword match (~70-80% precision)
- No es IA, es busqueda de strings: 'chat', 'email', 'async', etc.
- Usuarios Pro esperan precision. Falsos positivos = cancelaciones
- A mejorar post-lanzamiento con ML o revision manual

## Evaluacion del modelo de negocio

### Lo que SI funciona
- El dolor es real (15-20% poblacion tiene ansiedad telefonica)
- No hay competidor con filtro "no-phone"
- Costes operativos minimos (<$50/mes)
- Freemium es modelo probado (FlexJobs cobra $15/mes sin tier gratis)

### Lo que es problematico
- Revendes datos de terceros (Indeed, Glassdoor) — riesgo legal si creces
- Barrera de entrada cero — cualquiera replica esto en un fin de semana
- $9.99/mes por listings que estan gratis en Indeed — el valor real es solo el filtro
- Job seekers son usuarios temporales (1-3 meses) — churn alto
- Proyecciones del business plan ($80K ARR) son 3-4x optimistas

### Proyeccion realista Year 1
| Canal | Estimacion realista |
|-------|-------------------|
| Pro subs ($9.99/mes) | $500-2K/mes (50-200 subs) |
| Afiliados (resume services, cursos) | $500-2K/mes |
| Ads (AdSense en tier gratis) | $100-500/mes |
| Employer posting (futuro) | $0 Year 1 |
| **Total realista** | **$5K-20K Year 1** |
